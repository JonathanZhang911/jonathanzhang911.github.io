blueimp/jQuery-File-Upload@911dcf1

https://github.com/blueimp/jQuery-File-Upload

  1. Create a local folder "files-to-upload" and create 150 small text files in it
  2. Open OC v10.0.9
  3. Create a folder "test 1" and enter it
  4. Click upload and select all files in "files-to-upload" => all 150 files uploaded
  5. Create a folder "test 2" and enter it
  6. From file manager, drag and drop all files from "files-to-upload" onto the OC files panel => all 150 files uploaded
  7. Create a folder "test 3" and enter it
  8. From file manager, drag and drop only the folder "files-to-upload" itself onto the OC files panel => all 150 files uploaded in folder

I've also done another quick test with Firefox 61.0.1 (64-bit) also on openSUSE: works as well.
